## ss-angular

### Installation

You need Gulp installed globally:

```sh
$ npm i -g gulp
```

```sh
$ git clone git@github.com:art2cool/ss-angular.git
$ cd ss-angular
$ npm install
```


### Development

Want to contribute? Great!
Open your favorite Terminal and run these commands.

First Tab:
```sh
$ gulp watch
```
or
```sh
$ gulp test
```
